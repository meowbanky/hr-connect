## üìñ The Story

Hiring is usually a fragmented mess.[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)] Candidates jump through hoops, HR teams drown in spreadsheets, and data is scattered across five different "dashboards."[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]

**HR Connect** was built to solve the fragmentation problem.[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)] It isn't just a recruitment tool; it's a **Connection Engine.** It bridges the gap between high-potential talent and data-driven HR leadership by unifying the entire lifecycle‚Äîfrom the first click on a job board to the final onboarding checklist‚Äîinto one clean, high-performance interface.[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]

## üöÄ Getting Started

### Prerequisites

- PHP 8.2 or higher[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]
- Composer & NPM
- MySQL 5.7+[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)][[2](https://www.google.com/url?sa=E&q=https%3A%2F%2Fvertexaisearch.cloud.google.com%2Fgrounding-api-redirect%2FAUZIYQErxxLa8j0kBUx2MIgaFkZe2VBCDFmcU9-IDYvkYcwz-JNhuus-bijFLbSIYBAyCHI4CvxV5Pbhu1zBLjcJiEAqNKgN8IwuTUXlUxeLXH8LpuP9M5fR3mI%3D)]

### Installation

1. **Clone & Enter**

   ````bash
   git clone https://github.com/meowbanky/hr-connect.git
   cd hr-connect
   ```[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]

   ````

2. **Initialize Dependencies**

   ````bash
   composer install
   npm install && npm run build
   ```[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]

   ````

3. **Environment Setup**

   ````bash
   cp .env.example .env[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]
   # Update your DB_DATABASE, DB_USERNAME, and DB_PASSWORD
   ```[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]

   ````

4. **Database Migration**
   Import `database_schema.sql` into your local MySQL instance.[[1](https://www.google.com/url?sa=E&q=https%3A%2F%2Fgithub.com%2Fmeowbanky%2Fhr-connect)]

5. **Launch**

   ```bash
   php -S localhost:8000
   ```
